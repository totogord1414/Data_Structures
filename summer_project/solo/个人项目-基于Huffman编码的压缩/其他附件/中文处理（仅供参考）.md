在C/C++中的ASCLL码中：

- 一个英文字母（不分大小写）占**一个**字节的空间

- 一个中文汉字占**两个**字节的空间



因此可以考虑用unsigned char进行统一处理，以下代码及注释为压缩文件时的部分处理，供输入参考：

~~~c++
FILE* fp = fopen("D:/test.txt", "rb");//打开文件，以D盘中的test.txt为例
//读取文件内容，采用循环，一次读取1024个字节
unsigned char readbuff[1024];//设置缓冲区
int count[256];//记录字节出现信息
while (1)
{
    //用read_size来接收成功读取的字节数;fread函数的返回值表示成功读取的字节数量
    int read_size = fread(readbuff, 1, 1024, fp);
    if (read_size == 0)
    {
        break;//read_size = 0表示已经读到了文件末尾
    }
    //统计出现字符的次数
    for (auto i = 0; i < read_size; ++i)
    {
        //利用字符的ASCII码值作为数组的下标统计次数
        count[readbuff[i]]++;
    }
}
~~~



至于解压时的处理方式也类似，需要注意的是利用建好的哈夫曼树依次处理readbuff[]中的字节（一个字节八位，依次处理）。
